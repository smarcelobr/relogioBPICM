Projeto Relógio BICM

DOCUMENTO EM EDIÇÃO. NÃO ESTÁ PRONTO.

Este documento descreve o software e hardware usado no mecanismo do relógio da torre da Basílica do Imaculado Coração de Maria.

Componentes do Hardware

  • Redutor GSA 51
  • módulo ESP-12 (ESP8266 chip)

    • Software do ESP8266

    A programação do ESP8266 foi feita usando a linguagem LUA sobre um firmware NodeMCU customizado.

    - NodeMCU - Documentação - Documentação da Linguagem de Programação LUA

    Build do NodeMCU

    Utilizei o NodeMCU custom builds para criar um firmware customizado com os módulos necessários para controlar o mecanismo do relógio da Igreja.

    Os módulos escolhidos foram:

    • cjson
    • encoder
    • file
    • GPIO
    • http
    • net
    • node
    • pwm
    • sntp
    • tmr
    • uart
    • WiFi
